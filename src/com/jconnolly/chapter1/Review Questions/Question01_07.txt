Question:
Describe the history of Java. Can Java run on any machine? What is needed to run Java on a computer?

Answer:
    - Developed at Sun Microsystems in 1991.
        - Originally called "Oak" and developed for embedded chips in consumer electronics.
        - Renamed Java in 1995 and redesigned for developing internet apps.
    - Can be run anywhere.
    - You only need the Java Runtime Environment (JRE) to run Java but you need the Java Development Kit (JDK) to develop Java apps.
        - JDK includes compilers and debuggers.
