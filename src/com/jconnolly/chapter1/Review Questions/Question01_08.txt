Question:
What are the input and output of a Java compiler.

Answer:
Input: Java source code files which contain the code written by the programmer.
    - Typically files with a .java extension.

Output: Java bytecode files which contain the compiled code generated by the compiler.
    - These files have a .class extension and contain instsructions in the Java bytecode format.
    - Java bytecode format is a platform-independent representation of the Java source code.
    - Bytecode can be executed by JVM.
