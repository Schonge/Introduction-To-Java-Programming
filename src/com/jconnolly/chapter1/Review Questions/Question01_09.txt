Question:
List some Java development tools. Are tools like NetBeans and Eclipse different languages from Java or are they dialects or extensions of Java?

Answer:
Java Development Tools:
    - Eclipse is an Integrated Development Environment (IDE) that supports Java devlopment and provided features like code editing and debugging.
    - NetBeans is an IDE.
    - IntelliJ IDEA is an IDE.
    - Apache Maven is a build automation tool.
    - Gradle is a build automation tool.
    - JUnit is a Java testing framework.
    - Jenkins is an open source automation server for continuous integration and continuous delivery (CI/CD).

As listed above NetBeans and Eclipse are IDEs not different languages, dialects or extensions.
